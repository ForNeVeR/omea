Omea Reader Help: Working with Browser Plugins

Working with the Mozilla© Browser Plugin

This topic covers description of possibilities of Omea Connector plugin which works with such Internet Browses as Mozilla© and Mozilla©. Once you have installed Omea Connector plugin, you will be able to:

  • subscribe to RSS and ATOM feeds,
  • create Clippings (excerpts of the web pages or documents on the web),
  • annotate and categorize web pages

without need to switch to Omea and do it both when Omea is running and it is turned off.

Installing Omea Connector Plugin

Omea Connector Plugin is compatible with Mozilla© and Mozilla© Internet Browsers. The installation procedure for these two browsers is a little bit different.

Omea Connector plugin is included in the distribution of Omea and is installed in the same file folder where Omea is. So

To install the Omea Connector plugin into Mozilla©:

  1. Launch Mozilla© browser.
  2. From the File menu, select Open File.
    In the window which appears, navigate to Omea's installation folder, select the file named omeaconnector.xpi and click Open.
  3. In the Software Installation window which appears, click Install.
    The plugin will be installed and the Alert window opens saying that the Omea Connector installation was successful.
  4. Click OK in the Alert window and OK in the Software Installation window and restart Mozilla© so that your changes could take effect.

 

To install the Omea Connector plugin to Mozilla©:

  1. Launch Mozilla©.
  2. From the File menu, select Open File.
    In the window which appears, navigate to Omea's installation folder, select the file named omeaconnector.xpi and click Open.
  3. In the Software Installation window, click Install Now.
    The Extensions window opens with the list of plugins installed in Mozilla©.
  4. Close the Extensions window and restart the browser so that your changes could take effect.

Updating the Omea Connector Plugin

To update the plugin, download the latest version of Omea and repeat the procedure described above. You don't need to uninstall the plugin, and install it again to update it.

Subscribing to RSS and ATOM Feeds from within Mozilla© Browsers

Now you can subscribe to RSS and ATOM feeds right from Mozilla browsers without need to copy the URL and launch the Omea's Subscribe to Feed wizard. You have two options for creating the feeds subscription:

  • Subscribe to RSS or ATOM feed for the page that is currently opened
  • Create the feed subscription from the link on the web page: when you navigate to some link of the web page opened in the browser (for example, in the body of the page), you can subscribe to a feed which it may represent.

The procedure of subscribing to the feeds is the same both for Mozilla© and Firefox©:

  1. Launch Mozilla© or Mozilla© and navigate to some page.
  2. Do one of the following:
    • From the Tools menu, select Omea Connector > Subscribe to RSS/ATOM Feed for This Page
    • Click Subscribe to Feed button on the Omea toolbar.
    • Right-click the page and in the shortcut menu, select Subscribe to Feed for This Page in Omea

If Omea is running, the Subscribe to Feed wizard opens. The feed URL appears in the feed address field. Follow the wizard instructions (for example, type the feed name, select the feed type (RSS or ATOM) if necessary, select the feed you want to subscribe to if the page contains several feeds, and the feed group you want to add your feed to) to subscribe to the feed.

If Omea is not running, the Subscribe to Feed wizard appears next time you start Omea.

If the link on the web page represents a feed, you can subscribe to this feed using Omea Connector plugin:

  1. Launch Mozilla© or Mozilla© and navigate to some page.
  2. Right-click the link on the web page which represents the feed and on the shortcut menu, select Create Omea Feed Subscription from Link.

In the Subscribe to Feed wizard the feed URL appears in the feed address field. Follow the wizard instructions to subscribe to the feed.

If Omea is not running, the Subscribe to Feed wizard appears next time you start Omea.

Creating a Clipping

Clipping in general is an excerpt of text from any textual resource which can be used for the later reference. In terms of Omea Connector, Clipping is a piece of the web page content. Clippings themselves are a type of resource, and you can organize and search them in the same ways as other types you work with in Omea Reader Omea Pro.

You have two options for creating Clippings using Omea Connector plugin:

  • create the Clipping and edit its name and type some annotation for the Clipping.
  • create the Clipping from the selection and save the Clipping. In this case you don't have the possibility to edit the Clipping name and type some annotation.

To create a Clipping from the selection of the web page:

  1. Launch Mozilla© or Mozilla© and navigate to some page.
  2. Locate and select the text you want to clip and do one of the following:
    • From the Tools menu, select Omea Connector and then select either Create and Edit Clipping or Create and Save Clipping.
    • Click either Clip and Save or Clip and Edit button on Omea toolbar.
  3. Depending on which command you have selected Omea will behave the Clippings differently:
    • If you have selected the Clip and Edit or Create and Edit Clipping, the New Clipping window appears.
      • Type the Clipping name and annotation and click Save and Close.
        The Clipping appears in the Item Viewer of the selected resource tab in Omea. The title of the web page from which you have created a Clipping becomes the Clipping name (if you didn't edit it). The web page link is placed under the Clipping text.
    • If you have selected the Clip and Save, the Clipping will be saved in Omea and appears in the Clippings View of the Views and Categories pane when the All Resource tab is selected.

You can always access and view your Clippings in Clippings View of All Resources tab.

If Omea is not running, Clippings will be created next time you start Omea:

  • If you used Clip and Save command, the Clippings appear in the Clippings View of All Resources tab.
  • If you used Clip and Edit command, the New Clipping window opens.
    In this window you can edit the Clipping name and type its annotation.

Annotating the Web Page

With Omea Connector you can create the textual notes for the web pages. These notes like Clippings become a resource in Omea which is searchable and you can view them in the Annotated View of the Views and Categories pane.

To annotate the web page :

  1. Launch Mozilla© or Mozilla© and navigate to some page.
  2. Do one of the following:
    • From the Tools menu, select Annotate This Page.
    • Click Annotate and Categorize on Omea toolbar.
    • Right-click the web page or the link and on the shortcut menu, select Annotate This Page.

If Omea is started, the Edit Annotation window opens. Otherwise, you will be able to annotate the web page next time you start Omea.

Customizing Omea Connector Options

You can adjust such options as storing unprocessed requests and starting Omea when you use Omea Connector options while Omea is not running.

To configure these options in Mozilla©:

  1. Do one of the following:
    • From the Tools menu, select Omea Connector, and then select Options?
      Omea Connector Options dialog opens.
    • From the Edit menu, select Preferences.
      The Preferences dialog opens.
      • Click the JetBrains Plugins category. (If no subcategories are visible, double click the category name to expand it.)
        Omea Connector options appear in the right part of the Preferences window.
  2. Change the options available under the Store requests for later processing or select the Start Omea and retry request.

If Mozilla plugin is not installed, all Mozilla plugin options and actions are not available.

To configure these options in Firefox©:

  1. Do one of the following:
    • from the Tools menu, select Omea Connector, and then select Options?
      Omea Connector Options dialog opens.
    • from the Tools menu, select Extensions, select Omea Connector extension and click Options.
  2. In the Omea Connector Options window which opens, change the options available under the Store requests for later processing or select the Start Omea and retry request.
Depending on the option you have selected, your requests (subscribing to feed, annotating the web page, creating Clipping) will be either stored, or Omea will be started up immediately once you have tried to perform some Omea plugin action.

Uninstalling Omea Connector Plugin

You can uninstall the Omea Connector plugin from Firefox©:

  1. From the Tools menu, select Extensions.
  2. In the list of extension, select Omea Connector and click Uninstall.
  3. In the window that opens, confirm your wish to uninstall Omea Connector by clicking OK.
  4. Restart the browser so that your changes could take effect.

You cannot uninstall the Omea Connector plugin from Mozilla©.

Working with Internet Explorer© Browser Plugin

This topic covers description of possibilities of Omea plugin with Internet Explorer. Using this plugin you can:

  • subscribe to RSS and ATOM feeds,
  • create Clippings (excerpts of the web pages or documents on the web),
  • annotate and categorize web pages

without need to switch to Omea and do it both when Omea is running and it is turned off.

Compared with Omea Connector plugin for Mozilla©, you don't need to install the Omea plugin manually. The plugin it is installed onto your system when you install Omea. It is recommended to close Internet Explorer during the installation process.

Subscribing to Feed

You can subscribe to the feed which is located on the web page and the feed which represents a link on the web page from within Internet Explorer©, and you don't need to copy the web page or the feed link and launch the Omea's Subscribe to Feed wizard.

  1. Launch Internet Explorer© and navigate to some page.
  2. Do one of the following:
    • Select Subscribe to Feed button on Omea toolbar.
    • Right-click any place of the web page or the link on the web page and on the shortcut menu, select Subscribe to Feed.

If Omea is started, the Subscribe to Feed wizard opens. If Omea is not started, you will be able to subscribe to the feed next time you start Omea.

If the link on the web page represents a feed, you can subscribe to this feed using Omea Connector plugin:

  1. Launch Internet Explorer© and navigate to some page.
  2. Right-click the link on the web page which represents the feed and on the shortcut menu, select Create Omea Feed Subscription from Link.

In the Subscribe to Feed wizard the feed URL appears in the feed address field. Follow the wizard instructions to subscribe to the feed.

If Omea is not running, the Subscribe to Feed wizard appears next time you start Omea.

To configure these options in Internet Explorer©:

  1. Launch Internet Explorer©.
  2. From the Tools menu, select Omea Add-on Options?
    The Omea Connector Options dialog opens.
  3. Change the options available under the Store requests for later processing or select the Start Omea and retry request.
