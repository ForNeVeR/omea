Mozilla Omea Connector Plugin Installation

The Omea Connector plugin is an extension for Mozilla 1.7x and Firefox 1.x which allows you to create Omea resources directly from your browser.

  • Subscribe to RSS and ATOM feeds you're viewing in the browser;
  • Annotate and categorize the pages you're viewing;
  • Create clippings from the text you select on Web pages;
  • Synchronize bookmarks between Omea and Mozilla/Firefox.
